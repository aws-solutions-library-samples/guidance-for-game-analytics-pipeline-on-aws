The event library is a sample of custom event types for common industry use cases. The schema of the events is designed to flexibly accomodate a variety of game- and genre-specific details while having a standardized format that can be easily processed for use-cased focused insights. 

# User Tracking
Event types may contain a user identifier field to enable Player 360 insights that provide insights on per-player engagement. These identifiers must be unique identifiers to enable these insights.
